### Clone
```
git clone https://github.com/sethbergman/web-portfolio.git && cd web-portfolio
```
### Install dependencies
```
npm install
```
### Start the app
```
npm start
```
#### Visit [http://0.0.0.0:5000](http://0.0.0.0:5000) in your browser.
